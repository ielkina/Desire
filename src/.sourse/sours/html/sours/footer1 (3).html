События в браузере

Типы событий

  • clik - нажатие левой клавиши мыши
  • mouseover / mouseout - мыш наводиться на (покидает) элемент
  • keydown / keyup - нажатие / отжатие клавиши клавиатуры
  • focus - установка фокуса на элемент, напиример, input
  • submit - событие отправки формы
  • .....

Обработка событий

  • Через атрибут тега - onevent (например, onclick, onmouseover, onsubmit)
  • Вызов колбэка
  • Выборка ноды и добавление к ней событий onevent (по ссылке)
  • Использование "листнеров" addEventListener, removeEventListener

Доступ к содержимому ноды

  • Через контекст this
  • Через обьект события
